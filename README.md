## Features

* Select `HEX`and `RGB`colors.
* Convert color to `HEX`, `RGB`, `HSV`, `HSL`, `CMYK`.

## Install & Run

```
$ git clone https://github.com/cuongw/color-picker.git
$ cd color-picker 
$ npm install
$ npm start
```
